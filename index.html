Corpus Calculator

Use this tool to score a production file before quoting. It tells you the corpus number to enter in the platform and the correct installation time setting — in four steps.

1 File settings

Give the file a name and set the floor level. Floor adds 15 min per corpus for every level above ground — this applies across the whole file.

2 What's in this file?

Pick every product type the file contains — cabinets, benches, counters, planters, and so on. Each selection adds a card you fill in separately.

3 Fill in each product

On each card: enter the number of modules per size (S / M / L / Oversize), choose the install type, and mark unknown modules as on-site or unknown if the split isn't decided yet.

4 Set in platform

The calculator shows the total corpus number and the recommended installation type. Copy the backtrace block and paste it into the quote comment for a full audit trail.

Admin mode (top-right toggle) lets you adjust product types, size buckets, time constants, and images. Changes are saved in your browser.
